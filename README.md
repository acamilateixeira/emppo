## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/acamilateixeira/emppo

# Access
$ cd emppo

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
